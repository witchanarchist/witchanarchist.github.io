✹ The "Fuck Off" Web Manifesto ✹

Ten rules I wrote for myself, to remember why I build this website, and who it's for.


1. Effort Is the Gate, Not the Wall

If someone won't bother to read it, they don't need to understand.
If they can't paste an RSS link, they don't need updates.
If they won't write an email, their comment can wait.
This isn't a product. It's scribbles on my walls -
come closer only if it matters to you.


2. No One Is Watching, And That's the Point

I won't track. I won't analyze. I won't optimize.
You're anonymous here, just like everyone else.
If you're here, I hope it's because you found your way -
not because you were pushed.


3. Looks Don't Matter (But Soul Does)

I don't need slick graphics, dark mode, retina support.
If it's readable, it's enough.
I write each and every line of HTML with my own hands.
I write with what I have and I leave the rest behind.
If I made it, it's beautiful. For me.


4. Uptime Doesn't Matter

This site may go down.
That's okay.
Maybe the solar panel fails. Maybe the server dies.
That only proves it's real.
If I bring it back, it's because I care -
and if you come back, then maybe you do too.


5. The Web Is Not a Business

I'm not here to "serve content."
I write because I want to, not because I'm supposed to.
There is no target audience.
This is a space, not a service.


6. Dynamic Is Dead

I choose .html, .txt, raw files and rough edges.
No bloated frameworks. No needless scripts.
Every byte should carry meaning.


7. No One Gets to Own This

You can copy this. Mirror it. Print it. Archive it.
I don't believe in digital scarcity.
What's worth writing is worth keeping.
I might die today, but the internet should be forever.


8. Write Like It's a Letter

This isn't content. This is a message in a bottle.
Maybe to myself. Maybe to someone I'll never meet.
No hashtags. No SEO. Just words that needed to happen.


9. Build for the Few Who Get It

I don't need followers.
I don't need "engagement."
If two people find this in the dark and feel less alone - that's enough.
If no one does, it still mattered, because I needed to say it.


10. Labour Is Love

I build this by hand, because that's the only way it feels like mine.
I don't outsource the soul.
I don't outsource anything.
If it's ugly, it's honest.
If it's broken, I'll fix it - or leave it broken.
Either way, I'll love it, because it cost me effort -
and effort is love.


I don't owe the internet polish.
I don't owe the algorithm compliance.
This isn't a product.
This is my corner of the world -
and it will be the way I choose.
